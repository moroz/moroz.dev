This is the third installment of a series of articles discussing a cryptographic technique called _envelope encryption_. A more detailed description of this technique is provided in [part one](/blog/envelope-encryption-with-aws-kms-in-node-part-one/) of this series.

In [part two](/blog/envelope-encryption-with-aws-kms-in-node-part-two/), we introduced the AWS KMS API, created an IAM user, and set up local credentials. In this article, we are going to put it all together by writing a CLI tool to encrypt and decrypt files at will.
