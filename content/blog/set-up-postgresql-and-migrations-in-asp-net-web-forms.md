ASP.NET Web Forms is an old technology. As of this writing, the latest release is some 5 years old, and Microsoft, who brought this calamity upon us in the first place, has deprecated the technology by not including it by default in Visual Studio 2022.
When I tried to set up my very own Web Forms project on a fresh install of Windows 11, I quickly found out that I couldn't even get Visual Studio to show me the ASP.NET application template, and I had to resort to creating a starter project on another machine.
